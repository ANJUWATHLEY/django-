This is My About Page

Models (Data Structure)
Yaha humne models banaya h jo database ki table ki tarah hota h. Isme humne fields di h jaise name aur salary jo bhi user dega, yeh sab isme store hoga.

Views (Logic)
Models ka data yaha pe bheja jata h.
Templates (Design / HTML)
Frontend ka data likhte hain. Isme bhi hum base.html bana dete hain aur other files me isko import kar dete hain.
URLs
urls.py mein hum yeh define karte hain ki kaunsa view kis URL se chalega.
Admin Panel
Admin panel ko use karke tum apne employees aur departments ka data add/update/delete kar sakte ho. Models ko register karna zaroori hai admin.py mein taake wo admin panel mein dikhen.
Migrations (Database setup)
Jab tum koi naya field ya model add karte ho, toh tumhe migrate karna padta hai taake changes database mein apply ho jayein. Iske liye commands hain:
  • python manage.py makemigrations
  • python manage.py migrate
  • python manage.py runserver
Summary
  • Models: Database tables jisme fields hain (Employee, Department).
  • Views: Backend code jo data fetch karke template ko bhejte hain.
  • Templates: HTML design jo user ko dikhti hai.
  • URLs: Kis page par kaunsa view chalega.
  • Admin: Data ko manage karne ka backend.
  • Migrations: Database setup aur updates.
